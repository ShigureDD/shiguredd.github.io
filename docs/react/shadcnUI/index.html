SchadcnUI

Welcome to the shadcn-ui + Docusaurus integration example. This project demonstrates how to seamlessly integrate shadcn-ui components within a Docusaurus documentation site.

What is this?​

This is a collection of examples showing how to:

  • Set up shadcn-ui in a Docusaurus project
  • Use shadcn-ui components in MDX documentation
  • Create reusable component examples
  • Style and theme components to match your documentation

Getting Started​

Check out the Components section to see examples of shadcn-ui components in action. Each example demonstrates different aspects of the component library, from simple buttons to complex interactive UIs.
