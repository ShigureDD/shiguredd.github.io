# Basic

**Gross Amount** : Price \* Quantity

**Commission Fee**

**Accrued Interest**

**Stamp Duty**

**Withholding Tax**

### Transaction

> Buy : -1 \* Price \* Quantity + Commission Fee + Accrued Interest + Stamp Duty \+ Withholding Tax

> Sell : -1 \* Price \* Quantity + Commission Fee + -1 \* Accrued Interest + Stamp Duty + Withholding Tax

- Equity

- Fixed Income

- Real Estate

- Commodities

- Alternative Investment

- Other Asset
